Atributos HTML

Os atributos HTML fornecem informações adicionais sobre um elemento e são especificados no início da tag do elemento.

Exemplo de Atributos HTML

Aqui estão alguns exemplos de atributos HTML:

  • id - Identifica exclusivamente um elemento
  • class - Define uma ou mais classes para um elemento
  • href - Especifica o destino de um link
  • src - Especifica o caminho de uma imagem

Exemplo de Uso

Aqui estão alguns exemplos de como esses atributos podem ser usados em elementos HTML:

Exemplo de ID

<div id="conteudo">Conteúdo da Divisão</div>

Exemplo de Classe

<p class="destaque">Texto em Destaque</p>

Exemplo de Link

<a href="https://www.example.com">Link de Exemplo</a>

Exemplo de Imagem

<img src="imagem.jpg" alt="Descrição da Imagem">
