Blocos e Elementos Inline HTML

Os elementos HTML podem ser classificados em blocos ou elementos inline.

Exemplo de Elementos Block

Os elementos block começam em uma nova linha e se estendem pelo máximo possível na largura do seu container.

Exemplos de elementos block incluem:

  • <div>
  • <p>
  • <h1>-<h6>
  • <form>
  • <ul>
  • <ol>

Exemplo de Elementos Inline

Os elementos inline não começam em uma nova linha e ocupam apenas o espaço necessário em largura.

Exemplos de elementos inline incluem:

  • <span>
  • <a>
  • <strong>
  • <em>
  • <img>
  • <input>

Exemplo de Código HTML

                
                    <div></div>
                    <p></p>
                    <h1></h1>
                    <form></form>
                    <ul></ul>

                    <span></span>
                    <a href="#">Link</a>
                    <strong>Texto Forte</strong>
                    <em>Texto Emphatizado</em>
                    <img src="imagem.jpg" alt="Imagem">
                    <input type="text">
