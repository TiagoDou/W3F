Elementos HTML

Os elementos HTML são os blocos de construção das páginas da web. Eles definem a estrutura e o conteúdo de uma página.

Exemplo de Elementos HTML

Aqui estão alguns elementos HTML comumente usados:

  • <h1> - Título 1
  • <p> - Parágrafo
  • <a> - Âncora (Link)
  • <img> - Imagem
  • <div> - Divisão/Seção

Exemplos de Uso

Aqui estão alguns exemplos de como esses elementos podem ser usados em uma página HTML:

Exemplo de Título

<h1>Título da Página</h1>

Exemplo de Parágrafo

<p>Este é um parágrafo de exemplo.</p>

Exemplo de Link

<a href="https://www.example.com">Link de Exemplo</a>

Exemplo de Imagem

<img src="imagem.jpg" alt="Descrição da Imagem">

Exemplo de Divisão

<div>Conteúdo da Divisão</div>
