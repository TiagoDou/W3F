HTML Form Elements

Os elementos do formulário HTML são usados para coletar diferentes tipos de entrada do usuário.

Exemplos de Elementos de Formulário

Input Text

O elemento input type="text" é usado para campos de texto únicos.

                
                    <input type="text" name="username" id="username">

Input Password

O elemento input type="password" é usado para senhas.

                
                    <input type="password" name="password" id="password">

Textarea

O elemento textarea é usado para campos de texto longo.

                
                    <textarea name="message" id="message" rows="4" cols="50"></textarea>

Radio Buttons

O elemento input type="radio" é usado para botões de rádio.

                
                    <input type="radio" name="gender" id="male" value="male">
                    <label for="male">Masculino</label><br>

                    <input type="radio" name="gender" id="female" value="female">
                    <label for="female">Feminino</label><br>

Checkboxes

O elemento input type="checkbox" é usado para caixas de seleção.

                
                    <input type="checkbox" name="vehicle1" id="car1" value="car1">
                    <label for="car1">Carro</label><br>

                    <input type="checkbox" name="vehicle2" id="car2" value="car2">
                    <label for="car2">Moto</label><br>

Select Dropdown

O elemento select é usado para menus suspensos.

                
                    <select name="cars" id="cars">
                        <option value="volvo">Volvo</option>
                        <option value="saab">Saab</option>
                        <option value="fiat">Fiat</option>
                        <option value="audi">Audi</option>
                    </select>
