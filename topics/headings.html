Cabeçalhos HTML

Os cabeçalhos HTML são usados para definir títulos e subtítulos em uma página.

Exemplo de Cabeçalhos HTML

Aqui estão os diferentes níveis de cabeçalhos HTML:

  • <h1> - Título Principal
  • <h2> - Título Secundário
  • <h3> - Título Terciário
  • <h4> - Título Quaternário
  • <h5> - Título Quinário
  • <h6> - Título Senário
