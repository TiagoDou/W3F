HTML vs XHTML

HTML (Hypertext Markup Language) e XHTML (Extensible Hypertext Markup Language) são linguagens de marcação usadas para criar páginas da web.

HTML é uma linguagem de marcação mais antiga e leniente, enquanto XHTML é uma versão mais rigorosa e baseada em XML de HTML.

Diferenças Principais

  • HTML permite elementos sem fechamento e letras maiúsculas ou minúsculas em tags.
  • XHTML exige que todos os elementos tenham um fechamento e as tags devem estar em letras minúsculas.
  • XHTML é considerado uma aplicação XML e, portanto, exige que o documento seja bem formado (conforme as regras do XML).

Exemplo de Uso

Aqui está um exemplo básico de um documento HTML e um documento XHTML:

HTML

                
                    <!DOCTYPE html>
                    <html>
                    <head>
                        <title>HTML Document</title>
                    </head>
                    <body>
                        <p>This is an HTML document.</p>
                    </body>
                    </html>

XHTML

                
                    <!DOCTYPE html>
                    <html xmlns="http://www.w3.org/1999/xhtml">
                    <head>
                        <title>XHTML Document</title>
                    </head>
                    <body>
                        <p>This is an XHTML document.</p>
                    </body>
                    </html>
