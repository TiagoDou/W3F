IDs HTML

Os IDs HTML são atributos usados para identificar elementos HTML únicos em uma página.

Os IDs devem ser exclusivos em todo o documento HTML e são frequentemente usados para aplicar estilos CSS específicos ou selecionar elementos com JavaScript.
