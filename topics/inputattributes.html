HTML Input Attributes

Os atributos do elemento de entrada HTML fornecem funcionalidades adicionais e controle sobre os campos de entrada.

Exemplos de Atributos de Entrada

Value Attribute

O atributo value define o valor inicial de um campo de entrada.

                
                    <input type="text" name="username" id="username" value="john.doe">

Placeholder Attribute

O atributo placeholder exibe um texto de exemplo dentro do campo de entrada.

                
                    <input type="text" name="username" id="username" placeholder="Digite seu nome de usuário">

Disabled Attribute

O atributo disabled desativa um campo de entrada, impedindo que os usuários o modifiquem.

                
                    <input type="text" name="username" id="username" disabled>

Required Attribute

O atributo required torna um campo de entrada obrigatório, impedindo que um formulário seja enviado sem ele.

                
                    <input type="text" name="username" id="username" required>

Readonly Attribute

O atributo readonly torna um campo de entrada somente leitura, impedindo que os usuários o modifiquem, mas permitindo que ele seja enviado com o formulário.

                
                    <input type="text" name="username" id="username" readonly value="john.doe">
