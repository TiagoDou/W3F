HTML Input Types

Os diferentes tipos de entrada do HTML oferecem uma variedade de opções para coletar dados do usuário.

Exemplos de Tipos de Entrada

Input Text

O tipo de entrada text é usado para campos de texto únicos.

                
                    <input type="text" name="username" id="username">

Input Password

O tipo de entrada password é usado para senhas.

                
                    <input type="password" name="password" id="password">

Input Email

O tipo de entrada email é usado para endereços de e-mail.

                
                    <input type="email" name="email" id="email">

Input Number

O tipo de entrada number é usado para números.

                
                    <input type="number" name="quantity" id="quantity" min="1" max="100">

Input Date

O tipo de entrada date é usado para datas.

                
                    <input type="date" name="birthdate" id="birthdate">

Input Checkbox

O tipo de entrada checkbox é usado para caixas de seleção.

                
                    <input type="checkbox" name="vehicle1" id="car1" value="car1">
                    <label for="car1">Carro</label><br>

Input Radio

O tipo de entrada radio é usado para botões de rádio.

                
                    <input type="radio" name="gender" id="male" value="male">
                    <label for="male">Masculino</label><br>

Input File

O tipo de entrada file é usado para envio de arquivos.

                
                    <input type="file" name="myfile" id="myfile">
