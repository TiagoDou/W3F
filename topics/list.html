Listas HTML

As listas HTML são usadas para exibir itens em uma ordem específica ou não ordenada.

Exemplo de Lista Não Ordenada

Aqui está um exemplo de uma lista não ordenada HTML:

  • Item 1
  • Item 2
  • Item 3

Exemplo de Lista Ordenada

Aqui está um exemplo de uma lista ordenada HTML:

  1. Item 1
  2. Item 2
  3. Item 3

Exemplo de Código HTML

                
                    <ul>
                        <li>Item 1</li>
                        <li>Item 2</li>
                        <li>Item 3</li>
                    </ul>

                    <ol>
                        <li>Item 1</li>
                        <li>Item 2</li>
                        <li>Item 3</li>
                    </ol>
