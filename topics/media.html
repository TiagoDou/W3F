HTML Media Attribute

O atributo media é usado para especificar sob quais condições o link deve ser carregado.

Exemplo de Uso do Atributo Media

Aqui está um exemplo de como usar o atributo media em HTML:

Estilo para Impressão

O seguinte link de estilo só será usado quando a página for impressa:

<link rel="stylesheet" href="impressao.css" media="print">

Estilo para Tela Pequena

O seguinte link de estilo só será usado em dispositivos com largura de tela menor que 600 pixels:

<link rel="stylesheet" href="estilomobile.css" media="(max-width: 600px)">

Execução de Script em Tela Grande

O seguinte script só será executado em dispositivos com largura de tela maior que 1024 pixels:

<script src="script.js" media="(min-width: 1024px)"></script>

O atributo media pode ser usado em elementos <link> e <script> para especificar condições de mídia nas quais o recurso deve ser carregado ou aplicado.
